# Democracy Governance — Interactive Prototype

Presentation-quality React prototype for an AI Applications Platform — glassmorphism, spatial layout, and lightweight interactivity.

## Design

- Soft blue-white palette, light mode only
- Fixed top bar, left rail, bottom dock — no scrolling
- Symmetric 2×2 domain card grid with center hub
- Abstract UI blocks only (no dense dashboards)
- 🏛️ classical building as primary mark

## Four Domains

1. **Economy** — Governance Management Traceability
2. **Employment** — Information Curation Variety
3. **Ecology** — Intelligence Cooperation Integrity
4. **Education** — Inference Interaction Accountability

## Interactions

- Left rail or card click → select domain
- Bottom dock → switch workflow step (Choices / Preparation / Delivery)
- Generate App → build simulation with on-stage feedback

## Tech Stack

- React 19 · TypeScript · Vite · Bun
- Tailwind CSS v4 · Framer Motion · Lucide React

## Commands

```bash
bun install
bun run dev      # development
bun run build    # production build (use bun run, not bun build)
bun run preview  # preview dist/
```
